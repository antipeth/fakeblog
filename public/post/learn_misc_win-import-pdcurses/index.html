一.配置C++环境

1.下载MinGW

选用mingw。

首先，打开官方网站，选择最新的适合自己的版本下载。

如果不知道下什么，可以点此链接进行下载。 下载后，解压到非中文路径。

在例子中，我是直接解压到c盘下面。

2.添加环境变量

打开window设置。

点击关于(about)

点击高级系统设置(Advanced system settings)。

点击环境变量。

选中path那一行后，点击编辑。

点击新建，把你的mingw64\bin全部路径输进去。由于我的路径是C:\mingw64\bin，所以我添加的是C:\mingw64\bin。如果你的路径是C:\Progra Files\mingw64\bin，那你就添加C:\Progra Files\mingw64\bin。以此类推。

添加完成后点击确定。关闭剩下的窗口。

按Win+R键，输入cmd，点击确定，打开命令提示符窗口。

在命令提示符窗口中输入：

g++ --version

若又输出版本号，即成功配置c++环境。

二.导入PDCurses库

1.下载源码

访问官方下载网站)，找到最新的版本下载，不知道下什么可以点此链接下载。

下载后解压出来。

2.编译源码

进入wincon文件夹。

在此位置打开终端，如果不会，直接在上面的文件路径栏输cmd然后回车。

成功打开，且路径在这个文件夹下面。

输入：

mingw32-make -f Makefile

回车，等待代码滚完。

learn/pve/pve-hackintosh

3.将编译出来的东西添加到MinGW的库中

打开PDCurses-3.9\wincon这个文件夹，里面将里面一个叫pdcurses.a的文件复制到mingw64\lib目录下面，并重命名为名叫libpdcurses.a

打开PDCurses-3.9这个文件夹，把curses.h、curspriv.h和panel.h这三个头文件复制到mingw64\include目录下面。

至此导入成功，下面进行测试。

4.测试

找个地方创建test.cpp。

#include <cstring>
#include <curses.h>

int main(){
    initscr();
    raw();
    noecho();
    curs_set(0);

    char* c = "Hello, World!";

    mvprintw(LINES/2, (COLS-strlen(c))/2,c);
    refresh();

    getch();
    endwin();
    
    return 0;
}

然后在该目录下打开命令提示符。并输入：

g++ -o test test.cpp -lpdcurses

这里警告不要管他，已经编译成功了。在该文件夹下面生成了test.exe。

点击运行这个test.exe。

看到hello world，测试成功。PDCurses库可以正常使用。
